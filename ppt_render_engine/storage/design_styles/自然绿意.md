# 自然绿意 · 设计规范

> **设计原则**：每页必须包含 2 种以上视觉元素（色块、表格、箭头、菱形、流程图等），禁止纯文字堆砌。逻辑关系必须用图形表达。

## 配色

- **主色**: #70AD47（森林绿）
- **强调色**: #548235（深绿）
- **辅色**: #A9D18E（浅绿）、#4472C4（蓝色）、#ED7D31（橙色）
- **文字**: #000000 / #404040 / #FFFFFF

形状背景色使用上述色系，文字白色加粗。色块用圆角矩形包裹，多色组合避免单调。

## 模板

- 使用 **标准2** 模板
- 版式选用：标题幻灯片、节标题、标题和内容、两栏内容

## 字号体系

| 用途 | 字号(pt) | 粗细 |
|------|----------|------|
| 封面大标题 | 42 | Bold |
| 封面副标题 | 18 | Regular |
| 内页标题 | 28 | Bold |
| 节标题页标题 | 34 | Bold |
| 色块标题 | 18-22 | Bold |
| 色块描述 | 12-14 | Regular |
| 内页正文 | 14 | Regular |

## 逻辑图形模式

### 1. 步骤流程（横向）
圆角矩形横向排列，用 right_arrow 箭头连接：

```
[步骤1] → [步骤2] → [步骤3] → [步骤4]
```

配色递进：#70AD47（绿）→ #548235（深绿）→ #4472C4（蓝）→ #ED7D31（橙）

### 2. 分支判断（菱形）
diamond 形状（橙色 #ED7D31）表示条件，左右各一个 rounded_rectangle 表示结果。

### 3. 架构层级（纵向 y）
容器内纵向堆叠：顶层宽矩形（深绿）→ 中层（森林绿）→ 底层（浅绿），每层配文字。

### 4. 对比表格
3-4 列表格，表头加粗居中，行高均匀。用于技术对比、参数对照、优缺点比较。

### 5. 功能矩阵（2×2 / 2×3 色块）
容器内排列圆角矩形卡片，每张标题+一行描述。卡片间留 200000 EMU 间距。

### 6. 流程图
使用 flowchart_terminator（开始/结束，#70AD47）和 flowchart_process（步骤，#548235），arrow 连接。

## 容器模式

- 容器（children）绑定一个 placeholder，内部组合色块 + 文字
- 子元素坐标相对于容器，避免与其他元素重叠
- 典型用法：横幅标题 + 下方步骤列表；色块矩阵排列；层级架构堆叠

## 页面结构

1. **封面** — 标题幻灯片，标题+副标题居中
2. **节过渡页** — 节标题版式，大标题居中+描述文字
3. **内容页（架构）** — 标题和内容，容器内 3 层绿色矩形堆叠（标题层/内容层/数据层）
4. **内容页（流程）** — 两栏内容，左栏箭头串联步骤流程，右栏对比表格
5. **内容页（功能矩阵）** — 标题和内容，容器内 2×3 彩色圆角矩形卡片
6. **内容页（决策分支）** — 标题和内容，容器内菱形判断 + 两路分支结果
7. **结尾** — 标题幻灯片，结束语
